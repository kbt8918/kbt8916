## 1. 시장 현황

### 1.1 국내 고령화 현황

| 지표 | 수치 | 출처 기준 |
|------|------|-----------|
| 65세 이상 고령 인구 비율 | 19.2% (2024년) | 행정안전부 주민등록 인구통계 |
| 독거노인 수 | 약 202만 명 (2023년) | 통계청 |
| 치매 환자 수 | 약 100만 명 (2024년) | 중앙치매센터 |
| 치매 환자 연평균 증가율 | 약 5% | 중앙치매센터 |
| 초고령사회 진입 예상 시점 | 2025년 (65세 이상 20% 초과) | 통계청 전망 |

- 한국은 세계에서 가장 빠른 속도로 고령화가 진행 중이며, 2030년에는 65세 이상 인구 비율이 25%를 초과할 것으로 전망된다.
- 독거노인과 치매 환자 증가로 가족의 안전 모니터링 수요가 급속히 늘고 있다.

### 1.2 가족 위치 공유 서비스 시장 규모

| 구분 | 규모 | 비고 |
|------|------|------|
| 글로벌 Family Safety 앱 시장 (2023) | 약 3.2억 달러 | Grand View Research 추정 |
| 글로벌 시장 연평균 성장률 (CAGR) | 약 12.4% | 2024~2030 전망 |
| 국내 노인 케어 IT 서비스 시장 (2024) | 약 1,800억 원 | KCA 추정 |
| 스마트폰 보유 고령자 비율 (60대 이상) | 82.0% (2023년) | 과학기술정보통신부 |

- 60대 이상 스마트폰 보유율이 80%를 넘어서면서, 브라우저 기반 서비스 접근성이 크게 향상됐다.
- 별도 앱 설치 없이 웹 브라우저만으로 동작하는 서비스 형태는 고령자 진입 장벽을 낮추는 핵심 차별점이다.

### 2.4 경쟁 분석 시사점

- 네이버/카카오의 위치 공유는 임시 공유 방식으로, 지속적인 모니터링에 적합하지 않다.
- Life360은 기능이 풍부하지만 앱 설치 필수이며 한국어 지원이 불완전하고, 고령자를 대상으로 설계되지 않았다.
- 이동통신사(KT, SKT) 서비스는 자사 고객 대상 번들 서비스로, 통신사 가입자 외에는 진입 장벽이 있다.
- **웹 기반 + 고령자 특화 + 명시적 동의 체계**는 국내에서 공백 포지션이다.

## 3. 타겟 고객 분석

### 3.1 페르소나

#### 페르소나 A: 위치 확인자 (자녀/가족)

| 항목 | 내용 |
|------|------|
| 연령대 | 30~50대 |
| 직업 | 직장인 (부모와 별거 거주) |
| 상황 | 부모님이 치매 초기 또는 독거 생활 |
| 니즈 | 퇴근 후 또는 출근 중에도 부모님 위치를 빠르게 확인하고 싶음 |
| 불만 | 전화해도 안 받을 때 불안, 별도 앱 설치를 부모님께 부탁하기 어려움 |
| 기기 | PC + 스마트폰 모두 사용 |

#### 페르소나 B: 위치 제공자 (부모님)

| 항목 | 내용 |
|------|------|
| 연령대 | 65~80세 |
| 상황 | 스마트폰 사용은 가능하나 앱 설치/설정에 어려움을 느낌 |
| 니즈 | 자녀가 걱정하지 않도록 위치를 알려주고 싶지만 사생활도 지키고 싶음 |
| 불만 | 복잡한 앱 설정, 배터리 소모 불안, 개인정보 불안 |
| 기기 | 안드로이드 스마트폰 (갤럭시) 주 사용 |

### 3.2 고객 세그먼트

| 세그먼트 | 규모 추정 | 특징 | 우선순위 |
|----------|----------|------|----------|
| 치매 초기 부모님을 둔 가족 | 약 50만 가구 | 즉각적 모니터링 필요, 비용 지불 의향 높음 | 1순위 |
| 독거노인 가족 | 약 120만 가구 | 일상 안전 확인 목적, 무료 서비스 선호 | 2순위 |
| 고령 부모님과 원거리 거주 가족 | 약 200만 가구 | 주기적 확인 목적, 간헐적 사용 | 3순위 |

### 3.3 고객 구매 행동 패턴

- 앱스토어 검색보다 **네이버/구글 검색**으로 서비스를 발견하는 비율이 높다.
- 자녀가 설치 후 부모님에게 URL만 공유하는 형태의 온보딩이 선호된다.
- 무료 체험 후 알림 기능 필요 시 유료 전환하는 Freemium 모델이 적합하다.

## 4. 시장 기회 및 위험 요소

### 4.1 시장 기회

| 기회 요인 | 설명 |
|-----------|------|
| 초고령사회 진입 가속 | 2025년 초고령사회 공식 진입 → 노인 안전 서비스 수요 폭발적 증가 |
| 앱 설치 기피 트렌드 | 고령자뿐 아니라 전 연령대에서 앱 설치 피로도 증가, PWA·웹 서비스 선호 |
| 개인정보 민감도 증가 | 명시적 동의 기반 서비스에 대한 신뢰도 차별화 가능 |
| 정부 정책 연계 가능성 | 노인 돌봄 정책(복지부), 스마트 돌봄 서비스 연계 B2G 시장 진입 가능 |
| 카카오맵 API 성숙 | 국내 최적화 지도 API 안정화, 중소 서비스도 쉽게 활용 가능 |

### 4.2 위험 요소

| 위험 요인 | 영향도 | 대응 방안 |
|-----------|--------|-----------|
| 대형 플랫폼(네이버/카카오)의 유사 기능 강화 | 높음 | 고령자 특화 UX와 명시적 동의 체계로 차별화 |
| 개인정보보호법 강화 | 중간 | 동의 철회 즉시 중단, 7일 이후 자동 삭제 정책 준수 |
| 배터리/데이터 소모 불안으로 인한 이탈 | 중간 | 갱신 주기 조정 기능, 절전 모드 옵션 제공 |
| 이동통신사의 노인 케어 서비스 무료화 | 낮음 | 통신사 비가입자도 사용 가능한 범용성 강점 유지 |
| 브라우저 GPS 권한 허용률 저하 | 중간 | 권한 요청 UI/UX 최적화, 가이드 콘텐츠 제공 |

## 5. 포지셔닝 전략

### 5.1 포지셔닝 맵

```
                고령자 특화
                    |
         [우리 서비스]
                    |
웹 기반 ---+--------+--------+--- 앱 기반
                    |
         [네이버/카카오]  [Life360/찾아봐]
                    |
                범용
```

- X축: 웹 기반(앱 불필요) vs 앱 기반
- Y축: 고령자/부모 특화 vs 범용

### 5.2 핵심 포지셔닝 메시지

> "앱 없이, 링크 하나로. 부모님이 어디 계신지 바로 확인하세요."

| 대상 | 메시지 |
|------|--------|
| 자녀 (위치 확인자) | "앱 설치 없이 URL만 공유하면 부모님 위치를 실시간으로 확인할 수 있습니다." |
| 부모님 (위치 제공자) | "자녀에게 위치를 알려주고 싶을 때, 동의 버튼 하나로 공유하고 언제든 끌 수 있습니다." |

### 5.3 차별화 포인트 요약

| 차별화 요소 | 구체적 구현 |
|-------------|-------------|
| 앱 설치 불필요 | 웹 브라우저(모바일 크롬/사파리)에서 바로 동작 |
| 고령자 최적화 UI | 큰 글씨, 단순한 ON/OFF 토글, 권한 안내 가이드 |
| 명시적 동의·철회 | 부모님 측에서 언제든 공유 중단, 동의 기록 투명 공개 |
| 갱신 주기 커스터마이징 | 10초~수 분 설정 가능, 배터리 소모 최소화 옵션 |
| 국내 지도 최적화 | 카카오맵 기반, 국내 주소·장소명 정확도 우수 |

## 6. 수익 모델 검토

| 모델 | 내용 | 적합성 |
|------|------|--------|
| Freemium | MVP 기능 무료, 안전구역 알림·이력 조회 유료 | 적합 (진입 장벽 최소화) |
| 구독형 | 월 3,000~5,000원 / 가족 그룹당 | 2차 출시 시 도입 검토 |
| B2G | 지자체 노인 돌봄 시스템 연동 | 장기 검토 (조달 절차 복잡) |
| 광고 기반 | 노인 관련 서비스 광고 | 부적합 (신뢰도 저해 우려) |

**MVP 단계 권장 모델**: Freemium (핵심 기능 무료 제공 → 사용자 확보 우선)

## 7. 결론 및 시사점

| 항목 | 내용 |
|------|------|
| 시장 매력도 | 높음 — 고령화 가속, 독거노인 증가, 스마트 돌봄 수요 확대 |
| 경쟁 강도 | 중간 — 대형 플랫폼은 부모 특화 서비스 부재, 이동통신사는 자사 고객 한정 |
| 진입 장벽 | 낮음 — 웹 기반 기술 스택 성숙, 카카오맵 API 활용 가능 |
| 핵심 성공 요인 | 고령자 UX 최적화, 명시적 동의 체계, 빠른 MVP 출시로 선점 효과 확보 |
| 리스크 | 대형 플랫폼의 유사 기능 추가 — 고령자 특화 깊이와 브랜드 신뢰로 방어 |

> 부모님 위치 확인 서비스는 국내 초고령사회 진입이라는 메가 트렌드와 웹 기반 서비스의 접근성 이점이 맞물리는 공백 시장을 공략한다. MVP에서 핵심 기능을 빠르게 검증하고, 안전구역·이력 기능을 유료화 레버로 활용하는 Freemium 전략이 적합하다.
