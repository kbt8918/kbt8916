# NLM-02: 2022 개인정보 주요 이슈 법령해석 사례 30선

> 원본 URL: https://www.pipc.go.kr/
> 출처: pipc.go.kr (개인정보보호위원회)
> 수집일: 2026-05-23
> NotebookLM 소스 유형: pdf

---

개인정보보호위원회 발간 실무 법령해석 가이드. 실무에서 헷갈리기 쉬운 개인정보 보호법 해석 기준 30가지 사례 수록.

주요 챕터 구성:
| 챕터 | 내용 |
|------|------|
| Ⅰ | 개인정보의 수집·이용 |
| Ⅱ | 개인정보의 제공 |
| Ⅲ | 개인정보의 파기 |
| Ⅳ | 영상정보처리기기의 설치·운영 제한 |
| Ⅴ | 고유식별정보 처리 제한 |
| Ⅵ | 민감정보 처리 제한 |
| Ⅶ | 가명정보의 처리 |
| Ⅷ | 개인정보의 안전성 확보조치 |

핵심 사례 (프로젝트 관련):
- Ⅰ-5: 서비스 약관 동의 외 별도 동의 수령 필요 여부 → 위치정보 수집·가족 제공 동의는 약관과 분리 설계 필요
- Ⅱ-5: 홈페이지 등을 통한 개인정보 공유의 적법성
- Ⅲ-1: 수집한 개인정보의 보유기간 및 파기 시점 → 위치 이력 7일 보관 후 자동 파기 정책 근거

프로젝트 적용 포인트:
- 회원가입 화면: 서비스 이용 약관 동의 + 위치정보 수집 동의 + 가족 제공 동의 분리 설계 (법적 요건)
- 데이터 생애주기 관리: 위치 이력 7일 보관 후 자동 파기 (개인정보 최소 보유 원칙)
- 위치정보 수집·이용 목적 명시: 가족 안전 확인 목적에 한정, 제3자 제공 범위 명확히 고지
- 개인정보보호법 준수를 위한 핵심 법령 해석 지침서로 기능명세서(#6) 및 시스템정의서(#11) 작성 시 참조
