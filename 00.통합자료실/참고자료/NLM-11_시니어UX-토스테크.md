50대 이상 시니어 사용자들이 디지털 서비스 이용 시 겪는 5가지 오인지 사례 분석 (토스 UX 리서치).

주요 내용 — 시니어가 어려워하는 UX 5가지:
1. 클릭 가능한 진입점 오인지: 텍스트/화살표가 클릭 가능한지 몰라 버튼으로 인식 못 함
2. 예시 이미지 혼동: 예시 이미지를 본인의 실제 데이터로 착각
3. 모션 그래픽 오해: 애니메이션을 무조건 따라 하려 함 → 실제 행동 가이드 불명확
4. 질문형 문구 혼란: "위치를 공유하시겠습니까?" 등 주체가 불명확
5. 스크롤 미시도: 화면 아래 숨겨진 콘텐츠를 인지하지 못하고 스크롤하지 않음

시니어 UX 설계 원칙:
- 버튼/진입점은 형태로 명확히 구분 (border, 배경색, 아이콘 등)
- 예시 이미지에 개인화 텍스트 사용 주의
- 모션 그래픽 대신 명확한 행동 안내 문구 사용
- 질문형 → 지시/안내형 문구로 교체 ("위치 공유 시작하기")
- 핵심 기능은 스크롤 없이 첫 화면에 노출

프로젝트 적용 포인트:
- S-01(위치 공유 화면): '위치 공유 ON/OFF' 토글을 첫 화면 상단에 명확한 버튼 형태로 배치
- 문구: "위치를 공유하시겠습니까?" → "위치 공유 시작하기" / "위치 공유 중지"
- 위치 공유 상태는 색상 + 텍스트로 이중 표시 (시니어 가독성 확보)
