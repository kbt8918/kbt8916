스마트폰과 구글 지도를 이용하여 위급 상황에서 정확한 위치를 신속하게 전달하는 방법 가이드.

주요 내용:
- 대상: 말로 위치를 설명하기 어려운 시니어/고령자를 위한 위치 전송 방법
- 구글 지도 실시간 위치(라이브 위치) 공유:
  - 구글 지도 프로필 > '위치 공유' 선택 → 공유 시간 설정 → 카카오톡/문자로 링크 전송
  - 상대방은 링크만 눌러도 실시간 위치 + 이동 경로 확인 가능
- 핀 지정 단발성 공유: 지도에서 현재 위치(파란 점) 또는 특정 지점 길게 눌러 핀 지정 후 문자 전송
- 안드로이드 긴급 공유(Emergency Sharing): 안드로이드 '안전(Safety)' 앱으로 비상 시 미리 지정한 연락처에 즉시 위치 전송
- 응급전화 자동 위치 전송(ELS): 119 발신 시 안드로이드 기기가 자동으로 정밀 위치 데이터를 긴급구조대에 전송

프로젝트 적용 포인트:
- 부모님 위치 확인 서비스의 핵심 사용 시나리오와 직결 (위급 시 위치 공유)
- 카카오톡 링크 공유 방식 → 가족 그룹 초대 링크 UX 설계에 참고
- ELS/긴급 공유 기능 → 2차 출시 '위급 알림' 기능 기획 시 참고
