2025년 기준 자녀 위치 추적 앱 5종 비교 분석. 편리성·기능성·종합 점수 기준 평가.

주요 내용 — 앱별 특징:

| 앱명 | 주요 특징 |
|------|-----------|
| 도와줘 | 실시간 위치 확인, 안심존 이탈 알림, 긴급 상황 알림톡 전송 |
| 지나리 | 심플 인터페이스(시니어/초보자 특화), APK 설치 지원 |
| Life360 | 실시간 위치 + 운전 기록(속도·급정거), 사고 감지, 가족 채팅 |
| 아이쉐어링 | 90일 이동 경로 저장, 흔들기 SOS 기능, 출도착 알림 |
| 파인드마이키즈 | 스마트워치 연동, 주변 소리 듣기, 강제 알람 기능 |

비교 기준: 편리성 + 기능성 + 종합 점수

프로젝트 참고 포인트:
- '지나리'의 단순 UI → 시니어 부모님 대상 접근성 높은 UI 설계 모델
- '도와줘'의 안심존 이탈 알림 → 2차 출시 안전구역 기능 벤치마킹
- '아이쉐어링'의 SOS 기능 → 향후 위급 알림 기능 기획 참고
- 기존 앱 대비 차별점: 별도 앱 설치 불필요, 카카오/구글 소셜 로그인, 부모님 대상 초간단 UX
