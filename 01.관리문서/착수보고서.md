## 1. 추진 배경 및 목적

### 1.1 추진 배경

고령의 부모님을 둔 가족들이 부모님의 현재 위치를 실시간으로 파악하기 어려운 문제가 존재한다. 특히 치매나 독거 어르신 등 안전 관리가 필요한 상황에서 가족이 빠르게 위치를 확인할 수 있는 수단이 없다.

기존 방법의 한계:
- 전화로 위치를 확인해야 하는 번거로움
- 별도 앱 설치가 필요한 서비스는 고령자 접근성이 낮음
- 상용 위치 공유 앱은 가족 특화 기능이 부족하거나 복잡함

### 1.2 목적

| 목적 | 내용 |
|------|------|
| 주목적 | 가족 구성원이 부모님의 현재 위치를 웹 브라우저에서 실시간으로 확인 |
| 부목적 1 | 부모님 동의 기반의 투명한 위치 공유 체계 구축 |
| 부목적 2 | 별도 앱 설치 없이 스마트폰 브라우저만으로 이용 가능한 서비스 제공 |

## 2. 서비스 범위

### 2.1 사용자 역할

| 역할 | 설명 | 주요 행동 |
|------|------|-----------|
| 위치 제공자 (부모님) | 스마트폰 브라우저에서 GPS 위치를 공유하는 대상 | 위치 공유 동의, ON/OFF 토글, 공유 주기 설정 |
| 위치 확인자 (자녀/가족) | 웹에서 부모님 위치를 조회하는 사용자 | 지도 조회, 가족 그룹 관리, 알림 설정 |

### 2.2 MVP 범위 (1차 출시)

| # | 기능 | 설명 | 우선순위 |
|---|------|------|---------|
| F-01 | 회원가입 / 로그인 | 소셜 로그인(카카오/구글) 또는 이메일 인증 | 필수 |
| F-02 | 위치 공유 (부모님) | 브라우저에서 위치 공유 동의 후 30초 간격 GPS 전송 | 필수 |
| F-03 | 실시간 위치 조회 (가족) | 지도 화면에서 부모님 위치 마커 표시 및 자동 갱신 | 필수 |
| F-04 | 가족 그룹 관리 | 초대 링크/코드로 가족 그룹 생성 및 구성원 관리 | 필수 |

### 2.3 2차 출시 범위 (MVP 이후)

| # | 기능 | 설명 |
|---|------|------|
| F-05 | 안전 구역 이탈 알림 | 설정 반경 벗어날 경우 이메일/브라우저 푸시 알림 |
| F-06 | 위치 이력 조회 | 최대 7일 위치 이력 확인 |
| F-07 | 다중 부모님 계정 지원 | 그룹 내 여러 부모님 계정 등록 |

### 2.4 제외 범위

- 네이티브 모바일 앱 (iOS/Android)
- 실시간 통화 / 영상 통화 기능
- 결제 / 구독 기능 (MVP 기준)

## 4. 기술 스택

### 4.1 표준 스택

| 영역 | 기술 | 선택 이유 |
|------|------|-----------|
| Frontend | React + 카카오맵 API | 국내 지도 정밀도 및 한국어 주소 검색 지원 |
| 실시간 통신 | Server-Sent Events (M1~M4) / WebSocket (M5+) | Vercel 서버리스 환경 호환성 |
| Backend | Node.js (Express) | REST API + 실시간 통신 처리 |
| Database | PostgreSQL (Supabase) | 사용자/그룹/위치이력 영구 저장 |
| Cache | Redis | 실시간 위치 데이터 캐싱 (30초 TTL) |
| 인증 | JWT + OAuth2 (카카오/구글) | 소셜 로그인으로 부모님 가입 허들 최소화 |
| Deploy | Vercel + Supabase | 1인 운영 최적화, 서버리스 통합 배포 |
| CI/CD | GitHub Actions | 자동 테스트 및 배포 파이프라인 |

### 4.2 Backend 배치 전략

| 단계 | 방식 | 조건 |
|------|------|------|
| M1~M4 | Next.js/Express API Routes → Vercel 통합 배포 | 현재 기본값 |
| M5+ | Express 서버 분리 → Render/Railway 별도 배포 | WebSocket 필요 또는 Lambda 한계 초과 시 |

## 6. 비기능 요구사항

| 항목 | 요구사항 | 비고 |
|------|---------|------|
| 플랫폼 | 웹 (PC + 모바일 브라우저 반응형) | 별도 앱 설치 불필요 |
